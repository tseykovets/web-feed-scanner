Page with Feeds

Verify the behavior when scanning a page that has declared web feeds.

Preconditions

  • The Web Feed Scanner browser extension is installed.
  • This page is open in the browser.

Steps

  1. Open the context menu.
  2. Press the "Show page feeds" item.

Expected Result

  • A new tab opens displaying a message indicating that there are 6 feeds for this page:
    • Feed in RSS format
    • Feed in Atom format
    • Feed in JSON Feed format (application/feed+json)
    • Feed in JSON Feed format (application/json)
    • Feed in RDF/XML format
    • Feed in CDF format
