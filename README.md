# Web Feed Scanner

A browser extension (WebExtensions Manifest V3) that detects declared RSS and Atom web feeds on the currently loaded page. It parses `<link>` tags and exposes found feeds in the context menu (right-click) for quick access.

## Install as Unpacked Extension

1. Go to the Extensions page via menu or by entering the direct page address in a new tab, for example, `chrome://extensions`
2. Enable the Developer mode toggle switch.
3. Click the Load unpacked button and select the `src/` directory.

To update after changes, go to the Extensions page and click the Reload button under the extension title.
